# Prello
## Instructions to run


   * Do the npm install in both directories (front_end and back_end)
   * Install Mongodb from [Here](https://docs.mongodb.com/manual/administration/install-community/)
   * Start Mongodb
   * Start back_end
   * Start front_end

### Ports being used
* Mongodb :2707
* Socket.io :8000
* Back End Api (Express) :8000
* React App :3000
